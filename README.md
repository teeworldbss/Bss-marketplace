# 🛡️ BSS Business Group v1.0.0

A WhatsApp-style marketplace web app powered by **Firebase frontend** and **Gemini backend/storage**.

## 🚀 Features
- Register/Login system
- Marketplace product listings
- AI Assistant (Gemini-powered)
- Dashboard for buyers/sellers/admins
- WhatsApp-style UI theme
